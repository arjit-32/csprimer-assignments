## Assignment-1: HTML Structure & Semantics
_Goal: Learn proper HTML document structure and semantic tags._

Tasks ->
1. Create a full HTML page using:
    - `<header>`
    - `<nav>`
    - `<main>`
    - `<section>`
    - `<article>`
    - `<footer>`
2. Add content for each.
3. Validate HTML structure.
